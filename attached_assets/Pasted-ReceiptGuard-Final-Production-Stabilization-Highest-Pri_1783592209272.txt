3. Warranty System

Do NOT remove the Warranty feature.

Complete it so it is production-ready.

Implement:

Automatic warranty detection

Detect warranty-eligible purchases from imported receipts.

Extract:
	•	product name
	•	merchant
	•	purchase date
	•	receipt
	•	warranty information if available

If warranty duration exists in the email or receipt:

Extract it automatically.

If no duration exists:

Estimate intelligently based on product category, clearly mark it as an estimated warranty, and allow the user to edit it.

Never pretend estimated values are confirmed.
